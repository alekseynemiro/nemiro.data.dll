DB2Class3 program designed to generate the classes based 
on the SQL Server table schemes. The program creates classes 
in programming languages Visual Basic .NET and C#. 

Classes are designed to work with ORM Nemiro.Data.dll. 
The Nemiro.Data.dll library issued separately.


SYSTEM REQUIREMENTS

* Windows XP/Vista/7 or later
* .NET Framework 4.0
* Microsoft SQL Server 2005, 2008, 2008R2, 2012 or later
* Active network connection to access the SQL Server
* Keyboard, Mouse
* Microsoft Visual Studio 2010 or later (not necessarily)


IMPORTANT: Program may need to access the network. 
Make sure your firewall settings are not blocking your 
connection for DB2Class3.
