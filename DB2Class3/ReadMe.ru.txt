Программа DB2Class3 предназначения для создания классов на основе таблиц SQL Server.
Программа создает классы на языках Visual Basic .NET и  C#.
В качестве ORM используется библиотека Nemiro.Data.dll, которая поставляется отдельно.

Программа является бесплатной, в том числе для коммерческого использования.
Подробные сведения об условиях использования смотрите в файле License.ru.txt


СИСТЕМНЫЕ ТРЕБОВАНИЯ

* Windows XP/Vista/7 или выше
* .NET Framework 4.0
* Microsoft SQL Server 2005, 2008, 2008R2, 2012 или выше
* Активное сетевое соединение для доступа к SQL Server
* Клавиатура, мышь
* Рекомендуется: Microsoft Visual Studio 2010 или выше

Для работы с удаленными серверами SQL Server необходим доступ в Интернет.
Убедитесь, что ваш фаервол (firewall) не блокирует программе доступ в Интернет.
