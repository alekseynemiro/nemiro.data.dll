SqlAdmin - методыNemiro.Data.dll

Тип SqlAdmin предоставляет следующие члены.

Методы
  ИмяОписание
ClearTable
Удаляет все данные из указанной таблицы. Используется инструкция TRUNCATE TABLE (быстрое удаление с минимумом записей в журнал).
CreateTable(DataSet, Boolean)
Создает таблицы, указанные в экземпляре DataSet.
CreateTable(DataTable, Boolean)
Создает таблицу в базе данных.
DeleteTable
Удаляет указанную таблицу из базы данных.
Dispose
Освобождает все ресурсы, занятые объектом.
Equals
Определяет, равен ли заданный объект Object текущему объекту Object.
(Унаследован от Object.)
Finalize
Позволяет объекту Object попытаться освободить ресурсы и выполнить другие операции очистки, перед тем как объект Object будет утилизирован в процессе сборки мусора.
(Унаследован от Object.)
GetAllTables
Возвращает список всех таблиц, которые есть в базе.
GetAllTablesName
Возвращает имена всех таблиц, которые есть в базе данных.
GetAllTablesSize
Возвращает размер всех таблиц базы данных.
GetHashCode
Играет роль хэш-функции для определенного типа.
(Унаследован от Object.)
GetSqlServerProperty
Возвращает значение указанного свойства экземпляра SQL Server.
GetSqlServerProprty
Возвращает значение указанного свойства экземпляра SQL Server.
GetSqlServerVersion
Возвращает версию экземпляра SQL Server.
GetTableColums
Возвращает список всех колонок указанной таблицы.
GetTablePrimaryKey
Возвращает имя основного ключевого поля таблицы.
GetType
Возвращает объект Type для текущего экземпляра.
(Унаследован от Object.)
MemberwiseClone
Создает неполную копию текущего объекта Object.
(Унаследован от Object.)
ResetIdentity
Сбрасывает в ноль значение числового счетчика у указанной таблицы.
SetIdentityValue
Устанавливает новое значение для числового счетчика у указанной таблицы.
TableIsExists(String)
Проверяет существование указанной таблицы в базе данных.
TableIsExists(String, String)
Проверяет существование указанной таблицы на сервере.
ToString
Возвращает объект String, который представляет текущий объект Object.
(Унаследован от Object.)
