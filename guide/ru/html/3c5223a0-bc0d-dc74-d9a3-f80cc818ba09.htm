Параметры
primaryKeyValue
Тип: System Object
Значение ключевого поля, по которому будут получены данные из базы.
cache
Тип: Nemiro.Data.Caching CacheInfo
Параметры кэширования, если требуется указать параметры отличные от значений по умолчанию для экземпляра SqlClient. Если NULL (Nothing), то будут использоваться параметры по умолчанию.
