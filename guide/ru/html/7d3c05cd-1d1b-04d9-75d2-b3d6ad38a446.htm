Nemiro.Data.Sql - пространство имёнNemiro.Data.dll

Пространство имен [Nemiro.Data.Sql] содержит классы для работы с базами данных SQL Server.

Классы
  КлассОписание
BaseObject
Базовый класс, реализующий объектно-ориентированный доступ к данным (Object-Relational Mapping, ORM).
SqlAdmin
Класс предоставляет доступ к методам управления базой данных SQL Server.
SqlClient
Основной класс для работы с базами данных SQL Server.
SqlType
Вспомогательный класс для работы с типами данных SQL Server.
