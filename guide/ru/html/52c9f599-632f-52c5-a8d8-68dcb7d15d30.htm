Параметры
cacheCustom
Тип: System Type
Тип пользовательского объекта кэширования (должен происходить от ObjectCache). Значение по умолчанию берется из параметра NeData:Sql:CacheCustom файла конфигурации.
cacheCustomArgs
Тип:  System Object 
Дополнительные параметры инициализации пользовательского кэша, будут переданы в конструктор. Значение по умолчанию берется из параметра NeData:Sql:CacheCustomArgs файла конфигурации.
cacheDuration
Тип: System Int32
Продолжительность кэширования результатов запроса, в секундах. Минус один - без кэширования. Ноль - использовать значение и файла конфигурации, параметр NeData:Sql:CacheDuration.
