Параметры
value
Тип: System.Data DataSet
Набор данных, содержащий список таблиц, которые нужно создать в базе данных.
includeData (Optional)
Тип: System Boolean
Добавить в созданные таблицы, находящиеся в них данные или нет. По умолчанию False - не добавлять.
Заметки
Если хотя бы одна таблица уже существует в базе данных, то это приведет к возникновению исключения. Проверка делается глобально, перед созданием таблиц.
