Parameters
connectionString
Type: System String
Connection string.
            For example: user id=;password=;data source=(local);initial catalog=
            Default value: LocalSqlServer (key name of the connection string in the.config file)
            
cacheDuration
Type: System Int32
Duration caching the query results (in seconds). Minus one - without caching. Zero - use the value of the configuration file parameter NeData:Sql:CacheDuration.
